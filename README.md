Planes Móviles Megaline

### **Descripción del Proyecto**

La compañía móvil **Megaline** busca reducir el uso de planes antiguos entre sus clientes. Para lograrlo, desea implementar un modelo de machine learning que analice el comportamiento de los usuarios y recomiende uno de sus nuevos planes: **Smart** o **Ultra**.

Se dispone de datos históricos de los clientes que ya migraron a estos nuevos planes (provenientes del proyecto de *Análisis Estadístico de Datos*). A partir de esta información, se desarrollará un modelo de **clasificación** capaz de predecir el plan más adecuado para cada cliente.

---

### **Tarea**

Entrenar y evaluar diferentes modelos de machine learning para determinar cuál logra la mayor precisión al recomendar el plan correcto (Smart o Ultra).

---

### **Objetivo**

Construir un modelo de clasificación con una **exactitud mínima de 0.75**, que permita a Megaline recomendar de forma automática el plan óptimo para cada cliente, optimizando así su retención y migración a nuevos servicios.
